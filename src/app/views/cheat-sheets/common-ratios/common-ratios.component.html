Description: First Item represents the type of machine thats expected to assemble the following components.
Following items represent the amount of machines needed making them. If there is no machine mixing (assemblers/chemical plants/furnaces) then ratios stay the same b/w assembly tiers.

For example: To make rails, the right ratio is 1 iron stick assembler feeding 2 rail assemblers.
Using the level 3 assemblers you can produce 10 rails every second.
