Overview

Factorio is a game in which you build and maintain factories.
This is a compendium of the most common Factorio facts as of v{{ APP_SETTINGS.data_version }}.

A huge thanks to to all the contributors on GitHub, as well as the community who made the previous cheat sheets and other resources; such as the Wiki, Reddit, and Factorio Discord from which this is all based on.
This is only a sliver of the info available, for more in depth information please visit those resources.
