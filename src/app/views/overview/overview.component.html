Overview

Factorio is a game in which you build and maintain factories.

This is a compendium of the most common Factorio facts.

Some notes:
  • Check the Wiki, Reddit, and Factorio Discord for more in depth information/discussion.
  • Any section that deal with ratios assume no modules and consistent assembly machines.
  • Ratio sections are intended only as a starting point, for advanced ratios use a calculator.
  • Any of the sections can be collapsed/expanded by toggling the top right corner of each.
  • Please report any errors/suggestions on Github, Discord, Reddit, or Twitter.
